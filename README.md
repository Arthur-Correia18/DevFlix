# 🎬 DevFlix

Projeto desenvolvido durante o curso do YouTube para criação de um clone da Netflix utilizando **ReactJS**.

O objetivo do projeto é aplicar conceitos modernos de front-end, como componentização, consumo de API e manipulação de estado, construindo uma aplicação real e responsiva.

## 🚀 Sobre o projeto

O **DevFlix** é uma aplicação web que simula a interface da Netflix, exibindo filmes e séries organizados por categorias.

Os dados são consumidos de uma API externa (como TMDB), permitindo renderização dinâmica dos conteúdos, incluindo trailers. Projetos desse tipo são comuns para treinar React e integração com APIs reais :contentReference[oaicite:0]{index=0}.

---

## 🧠 Funcionalidades

- 🎬 Banner principal com filme em destaque
- 📂 Listagem por categorias (ação, comédia, etc.)
- 🖼️ Carrossel de filmes
- ▶️ Reprodução de trailers ao clicar no filme
- 📱 Layout responsivo (mobile e desktop)
- 🔄 Dados dinâmicos via API

---

## 🛠️ Tecnologias utilizadas

- ⚛️ React.js
- 📜 JavaScript (ES6+)
- 🎨 CSS3
- 🌐 API de filmes (TMDB)
- 📦 Axios (requisições HTTP)
- ▶️ React YouTube (player de trailer)
