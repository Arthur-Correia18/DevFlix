## ⚙️ Funcionalidades

- 🎬 Listagem de vídeos por categorias  
- ⭐ Sistema de favoritos (salvo no navegador)  
- 🔎 Barra de pesquisa  
- 🎠 Carrossel interativo  
- 📱 Layout responsivo (mobile-first)  
- 🎥 Página de exibição de vídeos  
- 🔝 Botão de voltar ao topo

## 🛠️ Tecnologias utilizadas

- React.js  
- JavaScript  
- CSS Modules  
- React Router DOM
